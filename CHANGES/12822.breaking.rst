Added ``aiofastnet`` package to ``speedups`` extra. aiofastnet provides faster alternatives to the standard loop functions, which are used to run server or establish connections. This is breaking because ``transport.get_extra_info('ssl_object')`` now returns ``aiofastnet.SSLObject`` when speedups are installed, which may lack some methods. If you experience any issues that you think might be related to this change, you can try to disable ``aiofastnet`` by uninstalling aiofastnet package.
